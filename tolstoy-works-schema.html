Tolstoy Works — Metadata Schema

Reference document for all metadata fields used in YAML frontmatter and Supabase.

Project: tolstoy.life Convention: camelCase → snake_case (Supabase) Date: 2026-03-28

1. Core Identity

FieldTypeReq.Description
idstring✓Unique project slug, e.g. anna-karenina
titleEnstring✓English title
titleRustring✓Russian title in Cyrillic
titleAlternativesstring[]Working titles, variant spellings, other language titles
 
# Example
id: "anna-karenina"
titleEn: "Anna Karenina"
titleRu: "Анна Каренина"
titleAlternatives:
  - "Anna Karenine"

4. Authoring Locations

Array — one entry per distinct location where the work was written.

FieldTypeDescription
authoringLocations[].countrystringCountry name
authoringLocations[].regionstringOblast, province, or region
authoringLocations[].citystringCity or town
authoringLocations[].estatestringEstate or named property
authoringLocations[].buildingstringBuilding name or type
authoringLocations[].roomstringRoom name or type
authoringLocations[].notesstringFree text, e.g. dates or context for this location

5. Manuscripts

Array — one entry per known draft. Writing materials are recorded at the individual manuscript level.

FieldTypeDescription
manuscripts[].draftIdstringUnique ID within this work, e.g. ms-01
manuscripts[].draftNumberintegerOrdinal position
manuscripts[].draftLabelstringFirst draftIntermediate draftFinal draftFair copyPrinter's copy
manuscripts[].currentRepositorystringFull institution name
manuscripts[].repositoryCitystring
manuscripts[].repositoryCountrystring
manuscripts[].repositoryCallNumberstringShelf mark, fond/opis/delo, or archival reference
manuscripts[].scan.urlstringDirect URL to scan or image
manuscripts[].scan.sourceInstitutionstringInstitution providing the scan
manuscripts[].scan.licensestringe.g. CC-BY 4.0 · Public Domain
manuscripts[].scan.creditstringAttribution string
manuscripts[].scan.dateAccessedstringISO 8601: YYYY-MM-DD
manuscripts[].writingMaterials.penstringe.g. Steel-nib dip pen
manuscripts[].writingMaterials.inkstringe.g. Iron gall ink, black
manuscripts[].writingMaterials.paperstringe.g. Laid paper, cream, unlined, folio
manuscripts[].writingMaterials.notesstringFree text

6. Transcriptions

Array — one entry per known transcription. Transcriber drawn from a controlled list.

transcriberIdFull name
sophia-tolstayaSophia Andreevna Tolstaya

maria-tolstayaMaria Lvovna Tolstaya
nikolai-gusevNikolai Nikolaevich Gusev
leo-tolstoyLeo Nikolaevich Tolstoy (autograph)
otherOther — specify in transcriberName
FieldTypeDescription
transcriptions[].transcriberIdstringFrom controlled list above
transcriptions[].transcriberNamestringFull name (required when transcriberId is other)
transcriptions[].relationToAuthorstringe.g. wife · daughter · secretary · self
transcriptions[].currentRepositorystringFull institution name
transcriptions[].repositoryCitystring
transcriptions[].repositoryCountrystring
transcriptions[].repositoryCallNumberstringShelf mark or archival reference

transcriptions[].scan.sourceInstitutionstring

transcriptions[].scan.dateAccessedstringISO 8601: YYYY-MM-DD
